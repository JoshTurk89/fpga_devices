To use script.bat; it is needed to generate the tcl file from vivado. 

To generate this file, the user shall follow the following bullets: 

  * Once the project is done, go to "File->Project->Write Tcl". 
  * Change the path of "Outout File" for this root. 
  * Select the following "Copy sources to new project" and "Recreate Block Desings using Tcl". 

Check the information stated into "vv_2_star.py" to be correct.
